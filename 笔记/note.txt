动态添加表单里有图片上传，先封装图片上传组件，然后在成功上传图片函数里将file保存到vuex里并且传一个函数给父组件，
在父组件里获取这个方法，这是上传成功时触发的，可以拿到当前的Index值，并且可以在vuex中拿到图片的Url


Vue在IE或者部分低版本浏览器中显示不支持 assign 属性或方法,导致页面空白无法显示； 引入npm install --save @babel/polyfill  在webpack.base.conf.js中配置  entry: ["@babel/polyfill", "./src/main.js"]


由于类的方法都定义在prototype对象上面，所以类的新方法可以添加在prototype对象上面。
Object.assign方法可以很方便地一次向类添加多个方法。

class Point {
  constructor(){
    // ...
  }
}

Object.assign(Point.prototype, {
  toString(){},
  toValue(){}
});
